Capacitar Sempre

Docker

Fornece uma camada de abstração e automação para virtualização de sistema operacional.

Git

É um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

JPA

É uma API padrão da linguagem Java que descreve uma interface comum para frameworks de persistência de dados.

Scrum

É um framework de gerenciamento de projetos, da organização ao desenvolvimento ágil de produtos complexos e adaptativos com o mais alto valor possível, através de várias técnicas.
