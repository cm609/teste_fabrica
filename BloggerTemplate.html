Web Site Template

Introdução

O Web Site Template oferece um modelo de site na forma de blog para uso pelas coordenações de curso ou diretorias do IFMT, sendo que atualmente está sendo utilizado por três cursos do Departamento da Área de Informática (DAI) do campus Cuiabá. Dentre as funcionalidades e conteúdos presentes no modelo podemos elencar:

  • Espaço para organização e postagem de atas, portarias, chamadas de eventos ou outras notícias extensas;
  • Espaço para notícias curtas integradas ao Twitter;
  • Calendário para apresentar os compromissos e/ou eventos próximos;
  • Buscador por texto livre ou por ordem cronológica;
  • Indexação por palavras-chave;
  • Espaço de direcionamento para outras mídias sociais;
  • Espaço para divulgação de vagas de emprego/estágio.
