Plano Individual de Trabalho (PIT)

Introdução

Semestralmente os docentes dos institutos federais devem entregar o plano individual de trabalho (PIT), documento este que contém a discriminação das atividades que serão executadas semanalmente totalizando a carga horária semanal de cada docente.

O projeto automatiza o processo de forma a reduzir o tempo dedicado à elaboração, envio, recebimento e feedback do PIT.


Apresentação das Tecnologias Utilizadas

JWT (JSON Web Token)

É uma forma de autenticação em aplicações web bem conhecida na qual o usuário informa seus dados de acesso, e uma vez que os mesmos estejam corretos, o usuário recebe um token com tempo predefinido de expiração, e com ele pode utilizar a aplicação. Esta tecnologia foi utilizada nesta aplicação pois o servidor é uma API desacoplada da aplicação gráfica que o usuário acessa, e para não trafegar o usuário e senha em toda requisição feita ao servidor, a aplicação do usuário guarda este token de acesso e o informa em toda requisição ao servidor.

Spring Framework

Um dos frameworks mais conhecidos e utilizados por programadores Java, o Spring destacasse por possui soluções completas para o desenvolvimento de aplicações web. O Spring possui módulos para persistência de dados, segurança, dentre outras funcionalidades e por estes recursos foi escolhido para compor a arquitetura da aplicação.

Angular

O Framework angular é uma plataforma de desenvolvimento web de código-fonte aberto baseado em TypeScript, desenvolvido e mantido pela Google. Foi escolhido para o desenvolvimento da aplicação gráfica deste projeto, pois possui uma gama de funcionalidades para o desenvolvimento de SPA (Single Page Application).
